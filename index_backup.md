Welcome to the official website for the IEEE PES Task Force on Building Energy Modeling for Flexible Scheduling.

This initiative integrates mainstream learning modeling algorithms (such as PINN, LLM-based agents, etc.), visualizes modeling results on both building scale and city scale, and provides a standardized workflow to bridge the semantic gap between BIM and power scheduling algorithms.

### Repositories

| Repository | Description |
|------------|-------------|
| [Building Modeling](https://github.com/qili531011-del/BEMS-modeling) | Building Modeling Methods |
| [Building Dataset](https://github.com/qili531011-del/BEMS-dataset) | Building Dataset |
| [Agent](https://github.com/qili531011-del/BEMS-agent) | LLM-based agent for human-interactive modeling |

### Technical Report

Our technical report covers:
- Simulation-based modeling (EnergyPlus, PVsyst)
- Data-driven modeling (ML, NN, RC, PINN, transfer learning)
- Physics-informed learning
- Human-interactive modeling (LLM agents)
- Applications for flexible scheduling (MPC, RL)
